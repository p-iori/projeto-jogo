Pontos e traços
Assim que direciono minha nave para o planeta, começo a receber um sinal, através dos meus comunicadores de curta distância. Confuso e incrédulo, conecto os equipamentos necessários para escutar a mensagem. Se trata de Código Morse, um método de comunicação criado a milênios para permitir uma troca de mensagens entre dispositivos primitivos, atualmente, ele é utilizado apenas para transmissões de emergência, o que me deixa ainda mais preocupado.
*Você encontrou uma mensagem em código morse! Ouça ao sinal recebido e tente decodificar o resto da mensagem utilizando as instruções na página. Caso não seja possível ouvir a mensagem, ou decodificar o código morse, você poderá clicar no botão "Revelar reposta" para receber a resposta (não recomendado). Esse planeta contém um pedaço das coordenadas da frota humana.

O código Morse é um instrumento de comunicação simples. Ele funciona a base de sons com durações diferentes: Os sons mais longos são representados por traços, enquanto os sons mais curtos são representados por pontos, intervalos maiores entre os sons representam a mudança de uma letra para outra. Aparentemente esse sinal tem origem em uma sonda de comunicação emergencial alocada no planeta. Essas sondas, não tripuladas, são utlizadas caso os sistemas padrão de comunicação sejam cortados, elas são lançadas em direção à bases ou naves alidas e transmitem mensagens pré-programadas, pelo o que parece, essa sonda foi lançada em direção ao planeta e acabou caindo nele, mas continuou transmitindo.
